      java中将一个list的内容复制给另一个list之后，去新的list进行操作的时候，原来的list也会发生变化

      提示

      问题：java中将一个list的内容复制给另一个list之后，去新的list进行操作的时候，原来的list也会发生变化

      # 问题分析：

      # 案例1：

          public static void main(String[] args) {
        List<Map<String, Object>> source1 = new ArrayList<>();
        List<Map<String, Object>> source2 = new ArrayList<>();
        Map<String, Object>map1 = new HashMap<>(4);
        map1.put("a","1");
        map1.put("b","2");
        map1.put("c","3");
        map1.put("d","4");
        source1.add(map1);
        source2.add(map1);
        System.out.println("第一次source1："+source1);
        System.out.println("第一次source2："+source2);
        for(Map<String, Object>changeMap:source1){
            if ("1".equals(changeMap.get("a"))){
                map1.remove("a");
                map1.put("a","5");
            }
        }
        System.out.println("修改后source1："+source1);
        System.out.println("修改后source2："+source2);
    }

      public static void main(String[] args) {

            item.remove("a");
        });
        System.out.println("修改后source1："+source1);
        System.out.println("修改后source2："+source2);

      source1集合复制给source2集合，然后修改source1集合的数据，source2的数据也会随之发生同样的变化。

      # 原因：

      map的put实现的是浅拷贝，向list集合中添加map数据，或者将一个list的内容复制给另外一个list，所添加的数据或复制的数据，不是值，而是一些对象，像C语言的指针，传递的是地址，指向对象，操作其中一个对象的值，别的list集合获取被修改对象的值就是新值，不是旧值了。

      # 解决方法：

      1. 手动赋值，效率高，但是代码啰嗦。
      2. 序列化与反序列化，使用SerializationUtils的clone(Object obj)方法，要求拷贝的对象实现了Serializable，Map不行，使用HashMap即可。
      import org.apache.commons.lang3.SerializationUtils;
import java.io.Serializable;
import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;

public class Test1 {
    public static void main(String[] args) {
        List<HashMap<String, Object>> source1 = new ArrayList<>();
        HashMap<String, Object>map1 = new HashMap<>(4);
        map1.put("a","1");
        map1.put("b","2");
        map1.put("c","3");
        map1.put("d","4");
        source1.add(map1);
        List<HashMap<String, Object>> source2 = new ArrayList<>(source1);
        //解决方法 注意引入的包是org.apache.commons.lang3.SerializationUtils
        source1= (List<HashMap<String, Object>>) SerializationUtils.clone((Serializable)source1);
        System.out.println("第一次source1："+source1);
        System.out.println("第一次source2："+source2);
        source1.forEach(item->{
            item.remove("a");
        });
        System.out.println("修改后source1："+source1);
        System.out.println("修改后source2："+source2);
    }
}
